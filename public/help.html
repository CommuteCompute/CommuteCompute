Help & Documentation

Everything you need to set up and use Commute Compute

Overview

Commute Compute is a smart transit display that shows your personalized commute information on an e-ink screen. It tells you when to leave, shows real-time departures, and even calculates if you have time for coffee!

Quick Start

Get up and running in 5 minutes:

  1. Go to the Setup Wizard
  2. Enter your addresses - Home, work, and optional coffee stop
  3. Set your arrival time - When do you need to be at work?
  4. Get your webhook URL - Copy the URL generated at the end
  5. Configure your device - Paste the URL in your CC E-Ink or Kindle settings
Done!
That's it! Your display will start showing your personalized commute.

Supported Devices

CC E-Ink Devices (Official)

  • CC E-Ink OG (7.5") - 7.5" display, 800×480, plug and play
  • CC E-Ink Mini - 4.2" display, 400×300, compact size

Kindle Devices (Jailbroken)

  • Kindle Paperwhite 3/4/5 - 6" display, requires jailbreak

  • Inkplate 6/10 - ESP32-based, DIY friendly

See docs/hardware/DEVICE-COMPATIBILITY.md in the repository for the full guide.

Setup Wizard

The Setup Wizard guides you through configuration step by step.

Step 1: Addresses

Enter your home address, work address, and optionally a coffee shop along your route. The wizard will find nearby transit stops automatically.

Step 2: Journey Settings

Set what time you need to arrive at work. The dashboard will calculate when you need to leave.

Step 3: API Keys (Optional)

For live data, you'll need a free API key from Transport Victoria OpenData. Without a key, the system uses scheduled timetables.

Tip
Tip: You can skip API keys to test with demo data first, then add keys later.

Frequently Asked Questions

How do I get an API key? +

1. Go to opendata.transport.vic.gov.au
2. Create a free account
3. Navigate to "My API Keys"
4. Generate a new key
5. Paste it in the Setup Wizard

Do I need to jailbreak my Kindle? +

Yes, Kindle devices need to be jailbroken to display custom content. This voids your warranty but is safe when following proper guides. See the device compatibility guide in docs/hardware/ for instructions.

Why isn't my display updating? +

Common causes:
• WiFi disconnected - check device network settings
• Webhook URL incorrect - verify in device settings
• Server sleeping - first request may take 10-15 seconds
• Rate limited - wait a few minutes and try again

How accurate is the "time for coffee" feature? +

The CoffeeDecision engine factors in:
• Walking time to the cafe
• Coffee preparation time (configurable)
• Walking time to your transit stop
• Real-time departure data
• Buffer time for safety

It's designed to be conservative - if it says you have time, you really do!

Does this work outside Victoria? +

Currently, live data is only available for Victoria (Melbourne metro, regional V/Line). Support for NSW, QLD, and other states is planned for future releases. You can use demo mode to see how it works anywhere.
