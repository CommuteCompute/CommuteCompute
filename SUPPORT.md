# Supporting Commute Compute(TM)

Thank you for considering supporting Commute Compute(TM)! This project is developed and maintained by **Angus Bergman** as a passion project to make Australian public transport more accessible.

## What Your Support Enables

### Development
- **More transit authorities** -- expanding beyond Victoria to all Australian states
- **New device support** -- TRMNL X, more Kindle models, custom hardware
- **International expansion** -- potential support for transit systems worldwide
- **Performance improvements** -- faster refresh, better reliability

### Infrastructure
- **Testing devices** -- hardware for development and testing
- **API costs** -- Google Places, Mapbox for enhanced geocoding
- **Server costs** -- hosting for documentation and demos

### Community
- **Documentation** -- tutorials, guides, video content
- **User support** -- helping people get set up
- **Design improvements** -- better UI/UX

---

## Transparency

This is a **non-commercial, open-source project**. All donations go directly toward:

1. Development time and tools
2. Testing hardware (displays, devices)
3. API and hosting costs
4. Coffee (lots of coffee)
